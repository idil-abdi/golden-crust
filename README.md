# 🥐 Golden Crust – Landing Page

## 📌 Project Overview
**Golden Crust** is a responsive bakery-themed landing page built using **HTML5** and **Sass (SCSS)**.  
The project focuses on clean design, structured layout, and modern styling techniques to create an engaging user experience.

---

## 🚀 Features
- Fully responsive design
- Modern and clean UI
- Flexbox layout
- Semantic HTML5 markup
- Smooth visual hierarchy

---

## 🛠️ Built With
- **HTML5** – Content structure
- **Sass (SCSS)** – Modular styling
- **CSS3** – Compiled from Sass
